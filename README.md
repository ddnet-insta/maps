# maps

Selection of maps for all official ddnet-insta gametypes. Supporting both 0.6 and 0.7


This is a generic repository that is just something to get you started playing around
with all modes. There are more complete and maintained specialized repos:

- [maps-06](https://github.com/ddnet-insta/maps-06) pvp maps for ctf, gctf, ictf and deathmatch
- [maps-07](https://github.com/ddnet-insta/maps-07) pvp maps for ctf, gctf, ictf and deathmatch
- [fng-maps-06](https://github.com/ddnet-insta/fng-maps-06) fng maps for 0.6 connections 
- [fng-maps-07](https://github.com/ddnet-insta/fng-maps-07) fng maps for 0.7 connections 

---

Of course you can do whatever you want and use any map for any gamemode.
But here are some recommendations which maps fit which modes.
